## 引き継いだ内容

- r9 UI lock KERRY版をベースにする
- 下部ナビは使わず、画面下部固定の主要アクションを使う
- 初期画面のみ出勤/退勤ボタンを表示する
- ルート選択画面は初期表示でヘッダーのみ
- 白タイル/グレータイルを選ぶと、そのルートだけをキープ表示する
- 「想定外の乗換」を選んだ場合は白/グレータイルを薄表示にする
- 乗換完了後は到着待ち画面へ進む
- 到着待ち画面のタイムラインには会社到着予定も表示する

## ダイヤルール

- 動物園前から淡路へ向かう列車だけ採用する
- 天神橋筋六丁目行き、天六行きは除外する
- 会社到着予想は `淡路到着 + 17分` を基本ルールにする
